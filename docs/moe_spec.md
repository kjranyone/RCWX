# 萌声・ロリ声の F0（基本周波数）分析

> **背景リサーチメモ**。ランタイム設定ではない。  
> 実装パラメータは [moe_boost_feminization.md](moe_boost_feminization.md) と README の Moe Boost を参照。

## 一般的な声の F0 範囲

| カテゴリ | F0 範囲 (Hz) |
|----------|-------------|
| 成人男性 | 85–180 |
| 成人女性 | 165–255 |
| 子供（女児） | 250–400 |
| ロリ声（演出） | 300–500+ |
| 萌声（演出） | 250–400 |

## ロリ声の特徴（概算）

- F0: 350–500 Hz（平均おおよそ 400 Hz）
- 基本周波数が高い（子供の声域の模倣）
- フォルマントも高め（短い声道のイメージ）
- ピッチ変動が大きい
- 呼吸的発声（息成分、H1–H2 差）

## 萌声の特徴（概算）

- F0: 280–380 Hz（平均おおよそ 320 Hz）
- ロリ声よりやや低め（成人女性の可愛い声に近い）
- ソフト発声、語尾上げ、わずかなハスキー成分

## 分析のポイント

測定の例:

- Praat 等の音声分析
- F0 抽出（autocorrelation 等）
- フォルマント分析（F1, F2, F3）

キャラクター別の概算例:

| キャラタイプ | F0 平均 | 特徴 |
|-------------|---------|------|
| ロリ系 | 380–450 Hz | 高 F0、強いピッチ揺れ |
| ツンデレ | 300–350 Hz | 通常域、怒り時は 350+ |
| お姉さん系 | 220–280 Hz | 低め・安定 |
| マスコット系 | 400–600 Hz | 超高域・非現実的 |

## 声優テクニック（参考）

- 頭声で F0 を上げる
- 声道短縮イメージでフォルマントを上げる
- リラックスした喉で息成分を増やす
- 舌を前寄りにして声道を短く感じさせる

## 注意

- 個人差・マイク・環境で大きく変わる
- 地声 F0 と演じる声の F0 は異なる
- 長時間の超高域演技は声帯負担が大きい

## RCWX への接続

Moe Boost は **F0 コンター変形のみ**で、上表の「フォルマント上昇」は再現しない。  
レジスタ目標は strength に応じておおよそ 165–220 Hz 帯へ寄せる設計（男声入力向け）。
